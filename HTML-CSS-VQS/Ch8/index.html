Batting Average (BA):
The total number of hits divided by the number of at-bats.
Home Run (HR):
A fair hit that allows the batter to round all of the bases and cross home.
Runs Batted In (RBI):
Any run credited to a specific batter that results from a fair hit ball or base on balls.
